**OAR 3.x breaking-change surface (from Nexus changelog):**

- **3.0.0** — FUNCTIONS API + custom-condition API changes. Only affects SKSE
  plugins that extend OAR (Math Plugin, author-built condition DLLs). **None
  installed in this list.**
- **3.1.0+** — IdleTime bugfix, non-ASCII path crash fixes, IsActorBase reliability fix.
  No negative impact on audited replacers.

**Author statement (OAR Nexus page):** "Fully backwards compatible with any
animation replacer mods created for Dynamic Animation Replacer."

## Decision

**Proceed with OAR 3.1.6 update.** All six replacer dependents use native OAR
JSON or DAR legacy conditions only; no OAR-extending SKSE plugins are present;
DAR itself is absent.

**Post-update smoke test (in-game):** tavern cheering variety, child idle
gestures, cart-driver sit animation, torch sprint after unequip, flute bard
alignment, Serana crossed-arms idle in Dawnguard.
