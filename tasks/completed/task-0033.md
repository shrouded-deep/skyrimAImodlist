## Description

The user has set a new curation direction: this list is a **power fantasy**
build, not a survival/realism build. Survival Mode Improved (and its CC
Survival Mode tie-in) will be removed. This is a philosophy-level decision
affecting list-wide curation, on the same order as the Lux and Pandora
migrations — research and scope first, no removals executed without
explicit sign-off.

This also means the cheat-mod batch report (task-0032,
`modlist/cheat-mods-batch-2026-07-07.md`) needs revisiting: several mods
were marked "needs review" specifically because they clashed with
Survival Mode Improved (the four "Infinite * Out of Combat" mods, Handy
Crafting and Spells). If Survival goes, that specific objection no longer
applies — though each mod's other risk factors (script footprint,
maintenance status, redundancy) still stand independently and should not
be waved through just because the survival objection is gone.
