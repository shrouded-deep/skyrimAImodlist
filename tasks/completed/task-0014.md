## Description

`modlist/version-check-2026-07-05.md` lists SkyUI (Nexus ID 12604) as
"Current (Nexus page stale since 2017 — mod is stable)". That's now wrong
— the same Nexus page received a major community update in March 2026
("SkyUI 6", developed with doodlum), now at version 6.9+. It's the same
mod page/ID, not a fork, updated in place after 9 years of dormancy —
which is exactly why the version-check's spot-check approach missed it.

This is higher-stakes than a routine Tier 1 bump: SkyUI has near-universal
downstream dependency (many mods require or modify its menus), the
update changed the plugin to a light plugin (can reset MCM/favorites
settings on update), and it added Skyrim.esm as an explicit master. Don't
treat this as a drive-by version swap.
