## Description

Add BodySlide, Outfit Studio, and DIP as MO2 executables on the Lost Legacy instance.
These tools are present in Anvil but not in Lost Legacy.

- `BodySlide and Outfit Studio` → `BodySlide x64.exe`, `OutfitStudio x64.exe`
- `Dynamic Interface Patcher - DIP` → `DIP.exe`

Anvil donor paths (read-only reference):

- `D:\Skyrim AI Modlist\Anvil\mods\BodySlide and Outfit Studio\CalienteTools\BodySlide\`
- `D:\Skyrim AI Modlist\Anvil\mods\Dynamic Interface Patcher - DIP\DIP\DIP.exe`

Copy these mod folders into `D:\Skyrim\mods\`, enable them in the `Lost Legacy - Fork`
profile modlist, then add MO2 executables to `D:\Skyrim\ModOrganizer.ini` mirroring
Anvil's entries with `D:/Skyrim/` pathing. Backup `ModOrganizer.ini` before editing.
MO2 must be closed during ini edit.
