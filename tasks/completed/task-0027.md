## Description

Research which Anniversary Edition / Creation Club content should be
imported into the list, given the established **power fantasy** curation
direction (see `modlist/power-fantasy-audit-2026-07-07.md` and
`AGENTS.md` Curation Direction).

This is a research-only task — no installs. The intent is to identify
CC content that fits the list's direction, not to add everything
available. The soft dependency on task-0033 exists because knowing what
survival content is being removed clarifies whether any CC content in
that category is even a live question.

**CC Survival Mode is included, not excluded.** The base CC content
(`ccqdrsse001-survivalmode.esl`) is being retained — task-0034 removes
only Survival Mode Improved (the Nexus overhaul), not the CC base mod.
The CC Survival Mode can be disabled via Settings → Gameplay → Survival
Mode without being uninstalled, and some mods may require it as a master.
Import it (or confirm it is already present) as part of the full AE pass.
