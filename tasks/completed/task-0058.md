## Description

`modlist.txt`. Because `modlist.txt` is REVERSE-ordered vs the MO2 UI (top of file =
lowest priority, bottom = highest), appending put all ~63 additions at the **highest
mod priority** — below Keizaal's tool outputs, overriding `DynDOLOD Output` /
`TexGen Output` and all of Keizaal's asset tuning wherever files overlapped. The
Restored CC mod (task-0057) also landed in limbo above the first separator.

User caught this and directed: move all additions under the `Uncategorized` separator.

- Additions were identified positionally: everything ABOVE the first separator
  (`Uncategorized`) + everything BELOW the last separator (`Skyrim AE 1.6.1170`).
  Keizaal's own mods are all BETWEEN those two separators.
- Moved all 65 addition lines (VB stack, city stack, Spaghetti, Restored CC, Mysticism
  texture pack) to immediately BEFORE (ABOVE) `+Uncategorized_separator`, preserving order
  and +/- enabled state.
- Safety-checked: 925 lines preserved, 903 mod lines preserved, 0 duplicates.

separator owns the mods ABOVE it, so they landed in the SimonRim group. User caught it.
