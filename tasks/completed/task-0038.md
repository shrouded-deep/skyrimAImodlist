## Result

**Completed 2026-07-07** (corrected after MO2 separator mechanics clarified).

MO2 requires an **empty mod folder** in `Anvil/mods/` whose name **exactly matches**
the modlist entry (e.g. `[Successor Additions]_separator` + `+[Successor Additions]_separator`).
Without the folder MO2 drops the line on save. **Edit modlist with MO2 open, then F5**
— restarting MO2 closed regenerates modlist from internal state.

**File order vs display:** modlist order is reversed from MO2 UI. Separator lines go
**after** their mods in the file so the header displays **above** them visually.

**Layout (file order, low→high priority):**
```
… Dev Tools …
  [cheat mods]
  -Cheats_separator
  [MLO/WSU disabled]
  [68 imported CC]
  +[Successor Additions]_separator
  ---- FINISHING LINE ---_separator
  [finishing mods…]
```
