## Description

Install 134 mods from the donor AE import into the `Fixes & Tweaks` section of
the Lost Legacy Fork modlist. Mod folders are already copied to `D:\Skyrim\mods\`
with their original donor prefixes intact. This task renames them and integrates
them into the modlist.

**Profile:** `Lost Legacy - Fork` at `D:\Skyrim\`
**MO2 must be closed before editing modlist.txt or plugins.txt.**

### Source

Folders to process: all entries in
`D:\Skyrim AI Modlist\anvil-successor\modlist\exports\donor-ae-import-selection.json`
whose names begin with `005.`, `006.`, `007.`, `008.`, or `009.`

### Rename rule

For each folder in `D:\Skyrim\mods\` matching the prefix pattern:
1. Strip the donor prefix: remove leading `XXX.XXX ◉ ` (digits, dot, digits, space,
   circle bullet, space)
2. Remove `[NoDelete]` and any surrounding spaces if present
3. Rename the folder in place

### Modlist placement

Add each renamed mod (enabled, `+` prefix) to
`D:\Skyrim\profiles\Lost Legacy - Fork\modlist.txt` **above**
`-Fixes & Tweaks_separator`. Insert the whole group as a block — order within
the group should match the original numeric prefix order (005 first, 009 last).

### Plugins

For any mod that contains `.esp`, `.esm`, or `.esl` files, enable the plugin
in `D:\Skyrim\profiles\Lost Legacy - Fork\plugins.txt` (prefix `*`). Place new
plugin entries in load-order sequence — after the existing Fixes & Tweaks plugins
and before the first non-Fixes section plugin.

### FODMODs

If any mod folder contains `fomod\ModuleConfig.xml`, flag it in the Result —
do not run the FOMOD installer (that happens in task-0075). Just note which
mods need a FOMOD run.
