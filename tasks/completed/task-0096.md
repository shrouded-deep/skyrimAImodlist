## Description

MO2 reports 264 full-weight ESP/ESM plugins — 10 over the 254 hard cap. The list
will not launch correctly above 254. Need to bring the count to 253 or below
(leaving at least 1 slot headroom).

**Target:** ≤ 253 full-weight plugins.

**Tool: ESLifier** — install from Nexus and run against the Lost Legacy - Fork
profile. ESLifier scans the full load order, identifies all plugins that qualify
for ESL-flagging (pure overrides with no new FormIDs, and plugins with new FormIDs
that fit in the 2048-record ESL range after compaction), compacts FormIDs where
needed, and sets the ESL flag in bulk. It handles both simple cases (pure-override
patches like Fork - City Stack JKs Interior Patch.esp) and plugins needing FormID
renumbering in one pass.

houseCARL's `compact_plugin` was attempted first but cannot flag pure-override
plugins (no FormIDs to renumber), and one-at-a-time compaction is impractical
at this scale. ESLifier is the right tool for bulk work.

**Do NOT ESL-flag:**
- Master files (.esm) that other plugins heavily depend on
- Plugins with > 2048 new FormIDs
- Behavior-engine outputs (Nemesis, DAR/OAR outputs)
- DynDOLOD.esp / Synthesis.esp / xLODGen outputs

## Steps

1. Install ESLifier via MO2 or standalone
2. Point it at the Lost Legacy - Fork profile (`D:\Skyrim\`)
3. Run its scan — review the candidate list before applying
4. Apply ESL flags; verify MO2 full-weight count drops to ≤ 253
5. Run a quick MAST scan to confirm 0 missing masters after flagging

## Acceptance criteria

- [ ] ESLifier installed and run against Lost Legacy - Fork
- [ ] Candidate list reviewed before applying (don't blindly flag everything)
- [ ] Full-weight plugin count ≤ 253 after run
- [ ] MAST scan: 0 missing masters
