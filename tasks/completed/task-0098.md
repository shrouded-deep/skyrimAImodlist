## Step 1 — Copy mod folders

Run `scripts/import-selected-donor-mods.ps1` with the **force/overwrite flag**.
Donor versions overwrite identically-named existing folders — intentional for the
graphics section where donor mods may share names with LL mods.

---

## Step 2 — Strip prefixes and rename

For every copied folder: strip the `XXX.XXX ◉ ` prefix, remove `[NoDelete]` where
present. Folders whose stripped name ends in `_separator` are separator entries —
handle them accordingly in Steps 3–4.

---

## Step 3 — Graphics section replacement (prefixes 016–066, ENABLED)

### 3a — Disable existing Lost Legacy graphics mods

In `modlist.txt`, find all enabled (`+`) mod lines that sit above
`-Graphics Improvements_separator`. Disable them (change `+` → `-`). Leave the
`-Graphics Improvements_separator` line itself in place as an anchor — it will
become the bottom of the new graphics block.

### 3b — Build the donor graphics section above the existing separator

The donor graphics selection includes sub-separator entries (e.g. stripped name =
`Visual Foundations_separator`, `Mesh Fixes_separator`, etc.). For each:

1. Create the folder `D:\Skyrim\mods\<SeparatorName>\` if it doesn't exist
2. In modlist.txt, the sub-separators and their mods go **above**
   `-Graphics Improvements_separator`, in prefix-number order (016 first, 066 last)

Layout within the new graphics block (each sub-group):
```
+<mod A>
+<mod B>
-<SubSeparatorName>_separator   ← stripped folder name ending in _separator
```

Non-separator mods in each prefix group go above their group's separator entry,
in the order they appear in the selection JSON.

### 3c — Enable plugins

For any graphics mod containing `.esp`/`.esm`/`.esl` files, enable the plugin in
`plugins.txt` (prefix `*`). Place new graphics plugin entries after the existing
Fixes & Tweaks / Graphics plugin cluster.
