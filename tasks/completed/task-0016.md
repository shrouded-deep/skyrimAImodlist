## Description

Project New Reign - Nemesis (60033) has had no updates since Dec 2021.
The user has decided to transition this list to its successor, **Pandora
Behaviour Engine+** (133232, v4.2.0-beta as of Mar 2026). This is a
list-wide change — Nemesis's output touches every animation-mod-driven
behavior file, and Anvil currently ships pre-generated Nemesis output —
so this needs real research before any execution task touches the actual
install.

## Result

**Completed 2026-07-06.**

### Pandora compatibility model

- Pandora **replaces** Nemesis; they should not coexist in production.
- Reads **Nemesis patch format** (`Nemesis_Engine/mod/<id>/`) — same mod inputs.
- Registers **FNIS-style animlists** without FNIS.exe (Serana Hood mod).
- **Native creature support** — drop Nemesis Creature Behaviour Compatibility.
- **XPMSSE** requires new mod: Universal Behaviour Runtime — Auto Skeleton Patch (176724).
- DynDOLOD / Synthesis / ParallaxGen: **no** Nemesis dependency.

### Nemesis dependents found (15 patch inputs)

12 MO2 mods + 3 bundled in engine: USSEP Behaviour Patch, TDM (3 modules),
Barstool Exit, Grain Mill, Bards Ghostly, Weapon Switch behavior patch,
First Person Teleport Bug Fix, Horsemen Torch, Hammering, Serana Hood anim,
Retimed Hit Frame, Turning Animation, Proper Spell Cast Direction.

All **compatible or expected compatible** with Pandora. None need Pandora-specific
patches for this list (no TUDM-style edge cases).

### Stale output finding

`Anvil - Nemesis Output` last run 2025-07-04 — wrong instance path, only 7/15
patches applied. Migration must wipe and full-regen.

### Broken installs flagged

Super Fast Get Up, 1st Person Greatsword, Unique Animations Reworked, TDM Tail
Fix — empty/meta-only folders; reinstall or disable before execution.
