## Description

The user wants to replace the Lux lighting suite (Lux, Lux Orbis, Lux Via —
Nexus 113002/56095/63588) with Modern Lighting Overhaul 2 (MLO2, Nexus
160748). This is a deliberate philosophy change, not a bug fix: Lux does
deep, comprehensive cell-by-cell rework with a large compatibility-patch

the Pandora migration. Research and plan only in this task — do not
