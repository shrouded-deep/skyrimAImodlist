### Findings

### Root cause

The plugin overrides a vanilla GMST without including its EditorID subrecord. This is
a valid (if unusual) ESP construction that xEdit can write, but Mutagen requires EDID
on GameSetting records and hard-errors. The stub's practical effect is a single game
setting value tweak — likely a UI string used by Description Framework's MCM or
display system.

### Action

Disable `Description Framework - Keizaal Cut` mod in `Keizaal - Fork` profile only.
Plugin count delta: 719 → 718 active checked (ESL, so no full-weight slot change).
No follow-up patches needed. houseCARL exclusion warning will not appear on next session.

*Note: xEdit can inspect the GMST value directly if the exact tweak is ever needed —
read the raw bytes at offset ~80 in the file to find the float/int value.*
