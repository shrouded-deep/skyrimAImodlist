## Description

Using `modlist/baseline-plugins-2026-07-05.txt` as the source list, check
whether the installed mods are current, or whether newer versions exist
upstream (Nexus Mods primarily). This is groundwork for the eventual
successor list — knowing what's stale now is cheaper than discovering it
mid-curation later.

Not every mod needs individual checking — prioritize mods that are:
- Core frameworks or widely-conflicting mods (per task-0002's baseline
  scan, these carry the most downstream risk if outdated)
- Mods you can tell from the filename/version string are likely old
- Anything already flagged anywhere in `modlist/decisions.md`

## Acceptance criteria

- [ ] Cross-reference installed mod versions against their current Nexus
      (or other host) listings, prioritized as above rather than
      exhaustively checking all 350+ plugins in one pass
- [ ] For each mod checked, note: current installed version, latest
      available version, and whether the update looks safe/routine vs.
      a major version bump worth extra caution before pulling in
- [ ] Flag any mods that appear abandoned, removed from Nexus, or
      replaced by a spiritual successor
- [ ] Write findings to `modlist/version-check-2026-07-05.md`
- [ ] Add a summary entry to `modlist/decisions.md`
- [ ] This is a report only — do not queue update tasks yourself; leave
      that as a human decision once the report is reviewed
