## Description

The user wants Lux Via's road-lantern/bridge content kept despite the
broader Lux → MLO2 migration (task-0021) — they specifically like the
roadside detail it adds and don't consider it dependent on Lux's core
lighting philosophy. Before re-enabling, confirm this is actually true at
the plugin/master level, not just conceptually — Lux Via's masters and
patch relationships need direct verification since Lux Via.esp and
Lux Via - plugin.esp were removed alongside Lux/Orbis in task-0021.
