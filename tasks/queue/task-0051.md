## Description

Keizaal vanilla+ ships with no major city expansion mods. Install the city stack
researched and validated on the Anvil project (task-0044), adapting the patch layer
for Keizaal's specific base mods (Lux/Lux Orbis/Lux Via, Bruma, Wyrmstooth, Tamrielic
Distribution).

**Primary reference:** `tasks/completed/task-0044.md` — mod selection, FOMOD choices,
and install order are reusable verbatim. The patch layer needs Keizaal-specific audit.

**Do NOT use Nolvus city mods as a source.** Nolvus's city stack is built around SR
Exterior Cities (SREX) — almost every city mod there has a bespoke `-SR Exterior Cities
Patch` or `-Nolvus Integration` companion. These are not portable without SREX, which
we are not using.
