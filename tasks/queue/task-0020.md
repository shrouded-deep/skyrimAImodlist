## Description

The user wants to add SR Exterior Cities — a large addition moving all
five interior city zones (Whiterun, Windhelm, Solitude, Markarth,
Riften — confirm exact list) out into the open Skyrim worldspace. This
is a worldspace-modification mod, not a lighting or behavior-engine
