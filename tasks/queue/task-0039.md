## Description

The 72 CC plugins imported in task-0035 need a cleaning audit. CC
content is commonly shipped with ITMs (Identical To Master records) and
deleted references (navmeshes, base objects) that can cause in-game
issues. houseCARL can identify these directly at the Mutagen record
layer without running xEdit.

This task is **audit only** — identify and document what needs cleaning.
A follow-up task will handle the actual cleaning writes (forwarding
correct records into a new patch plugin, not editing CC files directly).

Scope: all 72 active CC plugins (`cc*.esm`, `cc*.esl`, `cc*.esp`).
Exclude the 10 implicit masters and any CC plugins that are currently
disabled in the profile.

- [ ] Confirm `load_order_status` reports `Anvil - Main Profile` before

- [ ] Query all CC plugins for ITMs (records identical to their master
