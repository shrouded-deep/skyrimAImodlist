## Context

As mods have been added throughout the build, FOMOD patch selections made at
install time are now stale — they were chosen against an earlier state of the
list. This pass re-runs every FOMOD installer in the Fork with full knowledge
of the current load order, ensuring all cross-mod patches are correctly selected.

This includes Lost Legacy's own FOMOD mods, which may have patches for mods we
introduced (city stack, settlements, JK's/Ryn's stack, etc.) that were not
selected when Lost Legacy was originally installed.

**Run this task before the UI swap and before task-0060 (Tier 2 regen).** FOMOD
re-runs can change which plugins are active, which affects LOD generation.

**Do NOT run the Norden UI swap before this task.** Norden UI's FOMOD
auto-detects installed mods — it must run after the patch pass so it sees the
complete final load order. The UI swap is a separate step that follows this one.

**Target profile:** `Lost Legacy - Fork`
**Tool:** MO2 — right-click each mod → Reinstall

---

## Step 1 — Audit: identify all FOMOD mods in the Fork

Claude Code will generate this list before you begin. For each mod folder in
`D:\Skyrim\mods\` that contains a `fomod\ModuleConfig.xml`, check whether it is
active in `modlist.txt`. The output should be a checklist of every active FOMOD
mod, grouped by section.

*(Claude Code to populate the checklist below before this task runs.)*

---

## Step 2 — Re-run each FOMOD in MO2

For each mod in the checklist:

1. In MO2, right-click the mod → **Reinstall**
2. Work through the FOMOD options with the current modlist in mind
3. Select patches for mods that are **active (`+`) in modlist.txt**
4. Deselect anything whose required mod is absent or disabled
5. Confirm the reinstall — MO2 will update the mod folder in place

Pay particular attention to patch hubs and patch collections (JK's Interiors
Patch Collection, COTN patch collections, Fortified Morthal Patch Collection,
RedBag's Rorikstead patches, etc.) — these are the highest-value FOMOD runs.

---

## Step 3 — Sync plugins.txt

After all reinstalls, enable any newly selected patch plugins in `plugins.txt`
and disable any that were deselected. MO2 may handle this automatically on
profile refresh — verify either way.

---

## Step 4 — MAST scan

Run missing-masters scan. Target: 0 missing masters. Record plugin count.
