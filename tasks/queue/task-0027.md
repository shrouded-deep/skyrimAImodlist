## Description

Research which Anniversary Edition / Creation Club content should be
imported into the list, given the established **power fantasy** curation
direction (see `modlist/power-fantasy-audit-2026-07-07.md` and
`AGENTS.md` Curation Direction).

This is a research-only task — no installs. The intent is to identify
CC content that fits the list's direction, not to add everything
available. The soft dependency on task-0033 exists because knowing what
survival content is being removed clarifies whether any CC content in
that category is even a live question.

**Pre-decided exclusion:** CC Survival Mode (`ccqdrsse001-survivalmode`)
is explicitly out of scope regardless of other findings — it is being
removed as part of task-0034, not considered for import.

## Acceptance criteria

- [ ] Confirm which AE/CC content packs are already present (enabled or
      disabled) in the profile vs. which are entirely absent
- [ ] For each absent CC pack, assess fit against the power fantasy
      direction: combat variety, new weapons/armour/spells, follower
      content, dungeon/quest content good; survival systems, economy
      restrictions, or fragility mechanics out
- [ ] Note any CC content with known compatibility issues with this
      list's major mods (USSEP, Community Shaders, Lux suite, Pandora,
      SPID/OAR ecosystems)
- [ ] Flag any CC content that has a Nexus mod equivalent already
      installed — redundancy is a reason to skip
- [ ] Produce a ranked candidate list (import / skip / already present)
      with brief reasoning for each
- [ ] Write findings to `modlist/ae-cc-import-research-2026-07-07.md`
- [ ] Add a summary entry to `modlist/decisions.md`
- [ ] Do NOT create an execution task — this is scoping only; human
      sign-off required before any CC content is imported
