## Spaghetti Cities AIO → modular migration

Anvil currently runs AIO plugins. The migration procedure:

1. **Download** the individual city ESPs (already on Nexus; same author):
   - Spaghetti's Cities — Whiterun (80778)
   - Spaghetti's Cities — Windhelm (82820)
   - Spaghetti's Cities — Markarth (81200)
   - Spaghetti's Cities — Solitude (80986)
   - Spaghetti's Cities — Riften (82228)
   - Spaghetti's Towns — Riverwood (85356) — for Riverwood Has Walls patch
   
2. **Disable** `Spaghetti's Cities - AIO.esp` in MO2 (do not delete).

3. **Enable** the six modular ESPs in its place.

4. **NavCut:** The NavCut addon (120793) is AIO-scoped. Check whether individual
   NavCut slices exist per city; if not, keep the AIO NavCut active (it targets
   the same cells and is safe with modular base mods).

5. **Palaces + Faction Halls AIOs:** Leave as-is — interior-only, no hold dependency.
